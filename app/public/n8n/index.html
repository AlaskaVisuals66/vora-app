Workflows n8n — Helpdesk

Importe esses fluxos no seu n8n. Em cada card há um link direto pro JSON e um botão pra copiar a URL.

1 · Auto-resposta fora do expediente

Recebe webhook do Evolution e responde "fora do horário" entre 18h e 08h (BR).

Baixar JSON

2 · SLA Watchdog (Evolution)

A cada 5 min checa se a instância helpdesk-main está conectada; se cair, tenta reconectar.

Baixar JSON

3 · CSAT após fechar ticket

Webhook recebe ticket.closed, espera 30 min e envia pesquisa de satisfação 1-5 ao cliente.

Baixar JSON

4 · Resumo diário 18h

Cron diário às 18h consulta o estado da instância e envia um resumo pro gestor.

Baixar JSON

Como importar no n8n

  1. Abra https://projeto-n8n-n8n.8w2toq.easypanel.host.
  2. Clique em Workflows → ⋯ → Import from File e selecione o JSON baixado.
    (ou crie um workflow vazio, abra o menu ⋯ → Import from URL e cole o link copiado.)
  3. Clique em Active no canto superior direito.
  4. Para fluxos com webhook (1 e 3), copie a URL gerada pelo nó Webhook e cole nos lugares indicados:
    — fluxo 1: como webhook adicional do Evolution.
    — fluxo 3: dispare via POST com body { "instance": "helpdesk-main", "client_phone": "5521...", "protocol": "HD-..." }.
